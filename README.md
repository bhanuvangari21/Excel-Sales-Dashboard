# 📊 Excel Sales Dashboard

An interactive Excel dashboard analyzing Superstore sales performance, profit trends, and regional revenue insights.

## 🔧 Skills Used

* Pivot Tables
* Pivot Charts
* KPI Cards
* Dashboard Design
* Data Cleaning
* Data Visualization

## 📈 Dashboard Insights

* West region generated the highest sales
* Technology category produced the highest profit
* Sales peaked during November and December

## 🛠 Tools Used

* Microsoft Excel

## 📂 Project Files

* `Superstore_Sales_Dashboard.xlsx`

## 🎯 Objective

The goal of this project was to practice Excel data analysis and create an interactive business dashboard using real-world sales data.
